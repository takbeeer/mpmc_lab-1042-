# 8086 Assembly Language Lab Programs

## Overview

This repository contains a collection of basic 8086 Assembly Language programs developed as part of the **CSE 360: Microprocessor and Microcontroller Lab** course.

The programs demonstrate fundamental Assembly Language concepts including:

* Keyboard Input and Output
* Arithmetic Operations
* Conditional Statements
* Looping
* Pattern Printing
* Bitwise Shift Operations
* Rotate Operations

All programs are written for the **8086 Microprocessor** and can be executed using **EMU8086**, **MASM**, or **TASM**.

---

## Software Requirements

* EMU8086
* MASM (Microsoft Macro Assembler)
* TASM (Turbo Assembler)
* DOS Environment / DOS Emulator

---

## Programs Included

### 1. Input and Display Character

Takes a character from the user and displays it on the screen.

### 2. Addition of Two Numbers

Adds two single-digit numbers entered by the user.

### 3. Subtraction of Two Numbers

Subtracts the second number from the first number.

### 4. Multiplication of Two Numbers

Multiplies two single-digit numbers using the `MUL` instruction.

### 5. Division of Two Numbers

Divides two numbers and displays the quotient using the `DIV` instruction.

### 6. Sum of N Numbers Using Loop

Calculates the sum of N user-entered numbers using the `LOOP` instruction.

### 7. Smallest Number Between Two Numbers

Finds the smaller number using comparison and conditional jump instructions.

### 8. Smallest Number Among Three Numbers

Determines the minimum value among three numbers.

### 9. Largest Number Among Three Numbers

Determines the maximum value among three numbers.

### 10. Triangle Pattern Printing

Prints the following pattern using nested loops:

*

**

***

****

*****

### 11. SHL (Shift Left)

Performs left shift operation using the `SHL` instruction.

### 12. SHR (Shift Right)

Performs right shift operation using the `SHR` instruction.

### 13. ROL (Rotate Left)

Performs rotate left operation using the `ROL` instruction.

### 14. ROR (Rotate Right)

Performs rotate right operation using the `ROR` instruction.

## Concepts Covered

* ASCII to Decimal Conversion
* Decimal to ASCII Conversion
* DOS Interrupt (INT 21H)
* Register Manipulation
* Arithmetic Instructions (`ADD`, `SUB`, `MUL`, `DIV`)
* Conditional Jump Instructions
* Loop Control
* Pattern Generation
* Bitwise Operations
* Rotate Instructions

---

## Learning Outcomes

After completing these programs, students will be able to:

* Understand the structure of an 8086 Assembly program.
* Perform keyboard input and screen output operations.
* Implement arithmetic calculations.
* Use decision-making instructions.
* Apply loops for repetitive tasks.
* Work with bitwise and rotate instructions.
* Develop problem-solving skills using low-level programming.
